# Portfolio Website

## Overview

Modern developer portfolio website built with TanStack Start, showcasing Arpit Nigam's professional experience, projects, and skills. The site features server-side rendering, file-based routing, and a clean, responsive design.

## Current Status

✅ **Complete** - Base portfolio with personalized content

## Architecture

- **Framework**: TanStack Start (full-stack React with SSR)
- **Styling**: Tailwind CSS v4
- **UI Components**: Custom components (Button, Card)
- **Icons**: Lucide React
- **Routing**: File-based routing with TanStack Router
